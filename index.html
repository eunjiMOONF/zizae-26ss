02 — PAST

25SS 회고

지재의 현 위치를 정의하다

노출과 가격은 비슷, 성과는 '전환(CVR)'에서 갈림

BEST는 "클릭 이후 확신" / WORST는 "클릭 이후 실망"

BEST 30 vs WORST 30 핵심 지표 비교

항목 BEST 30 WORST 30 결론
평균 가격38,714원37,537원가격이 원인이 아님
평균 노출68,94762,870노출이 원인이 아님

평균 CVR1.35%0.18%결정적 차이(상세에서 붕괴)

⚠️ 핵심 메시지: WORST는 "안 팔린 상품"이 아니라 '상세페이지 단계에서 죽은 상품'

왜 CTR이 아니라 CVR인가?

1,000명이 상품을 봤을 때 시뮬레이션

현재 WORST

CTR 3.00% → 30명 클릭

CVR 0.18% → 0.05명 구매

썸네일만 개선

CTR 4.68% → 47명 클릭

CVR 0.18% → 0.08명 구매 (×1.6)

상세페이지 개선

CTR 3.00% → 30명 클릭

CVR 1.35% → 0.41명 구매 (×8.2)

💡 결론: 썸네일을 BEST 수준으로 올려도 구매 1.6배. 상세페이지를 고치면 8.2배.

✓BEST 성공 공식

  • • 카테고리: 원피스가 매출 엔진(45~50%), 하의는 단품일 때만 강함
  • • 소재/핏: 링클·바스락 + 슬리브리스·밴딩·와이드 → '시원하고 편한 장점'
  • • 패턴/컬러: 리프·도트·플라워 스테디 패턴 + 뉴트럴/딥톤 '베이직 컬러'
  • • 디테일: 벨트/밴딩/퍼프/플리츠는 조합이 맞을 때 성과

✕WORST 실패 공식

  • • 전환 붕괴: CTR은 나와도 상세에서 확신을 못 줘 CVR 무너짐
  • • 미스매치: 상하의 셋업은 선택 부담으로 약세(BEST 0)
  • • 실패 신호: 레이스/시스루/프릴/핀턱/스티치/골지/셋업/트렌치
  • • 다양한 디테일의 조합 → 복잡·불확실성 → 이탈

25SS 액션

🟢 KEEP (유지)

  • • 링클/바스락/밴딩 소재 안정적 판매
  • • 원피스·스커트 단품 매출 중심
  • • 2SET/3SET 객단가 상승
  • • 트위드 자켓 단품 틈새 성공
  • • 재진행 상품 평점 4.46

🔴 STOP (중단)

  • • 상하의 셋업 상품 다수 실패
  • • 디테일 과잉 상품 판매 0
  • • 4~6만원 애매한 가격대
  • • 상세 부족 → CTR↑ CVR↓

🟡 FIX (개선)

  • • 3~5만원대 상품 확대
  • • 4만원↑ = 명확한 명분 필수
  • • 상하의 셋업은 검증 포맷만
  • • 핏/사이즈 기준 사전 정의
  • • 실패 키워드 사전 차단

원피스

저지드레이프랩배색벌룬
  • • 편안한 저지소재를 드레이프 디테일로 우아하게
  • • 미니원피스는 벌룬 디테일 더하기
  • • 경쟁사의 시그니처 디자인 가져오기
  • • 배색과 셔츠 디자인으로 포멀 라인 유지

블라우스 / 셔츠

오픈넥와이드카라보우타이쉬폰배색
  • • 오픈 와이드 카라로 시원하게
  • • 쉬폰 소재에 패턴 더하기
  • • 사이드 보우타이 디테일로 페미닌 무드 유지
  • • 몸에 붙지 않는 소재의 셔츠

니트 / 가디건

골지오픈넥플라운스배색
  • • 신축성 좋은 골지 소재로 편안하게
  • • 오픈넥 혹은 플라운스 디테일로 페미닌 무드 유지
  • • 기존 BEST 상품의 디테일 변형 상품 기획
  • • M품번 니트는 루즈하고, 편안한 디자인에 트렌드 포인트 더하기

스커트

쉬폰배색고데링클플리츠
  • • 헴라인 쉬폰 배색으로, 비침 우려없이 트렌드 패브릭 더하기
  • • 편안한 무드의 링클소재를 활용한 밴딩 스커트
  • • 바스락/새틴 소재에 풍성한 사이드 고데 디테일 / 플레어 디자인으로 우아하게
  • • 캐주얼한 데님 소재에 핀턱 디테일로 페미닌한 실루엣 유지
  • • 전체밴딩 / 뒷밴딩 디테일로 편안한 착용감

팬츠

하렘신축성린넨라이크빅패턴
  • • 트렌디한 하렘팬츠 디자인은 흐르는 듯 편안한 소재의 슬랙스
  • • 빅 패턴 와이드 플레어 팬츠의 다양한 베리 아이템
  • • 가볍게 찰랑이며 신축성 좋은 린넨라이크 소재
  • • 부츠컷은 기존보다 기장을 높여 가볍고 경쾌한 무드

SET (2SET/3SET)

상하의 X앙상블가디건set
  • • 상하의 셋업은 검증된 디자인으로 소량만 기획
  • • 사이즈 커버리지가 넓은 니트 소재 위주로 기획
  • • 단품으로도 충분히 활용이 가능한 상품 위주로

자켓

  • • 트위드 아이템은 극 소량만

핵심 키워드: 행거컷 카탈로그 무드 (자동화) / 얼굴 크롭컷 확대

행거컷 카탈로그 이미지가 모델컷 대비 CTR 성과가 우수함을 확인

  • • 모델컷: 브랜드 무드 전달에는 효과적이나, 리스트 탐색 단계에서는 정보 인지 속도가 느리고 시선 분산 발생
  • • 행거컷: 상품의 실루엣·색상·형태를 즉시 인지할 수 있어 클릭 유도에 유리한 구조

웹앱 핵심 기능

  • • 이미지 업로드 → 자동 분류(모델컷/상품컷)
  • • 행거컷 변환 자동 적용
  • • 배경 톤·그림자·여백 프리셋 적용
  • • 결과물 자동 저장 및 파일명 규칙화

CTR 개선 자동화 플로우

CTR 저조 상품 탐지 → 품번 리스트 생성 → 이미지 자동 수집 → 웹앱 이미지 개선 → 셀러 어드민 자동 반영 → 노출 썸네일 교체

💡 전략적 의미: 본 시스템은 단순한 이미지 편집 도구가 아닌, 이미지 성과 개선을 자동으로 반복하는 Self-healing System. 이미지가 직접 매출에 기여하는 구조를 브랜드 내부 자산으로 구축.
